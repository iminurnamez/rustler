Dependencies:

Requires Python 2.7 (untested on Python 3) and pygame (http://www.pygame.org/download.shtml)




Getting Started:

NOTE: This game has only been tested on Windows 8.1

R.U.S.T.L.E.R. can be run from the CLI like any other python program by entering "python rustler.py" from the game's directory.
You can also double-click on the rustler.py file icon to launch the game.
If you have downloaded an executable version double-click on the rustler.exe file icon to launch the game.



Playing the Game:

Use your horse and whip to drive the cows. Cows close to your horse will adjust their course to travel away from you.
Use your lasso to rope wayward cows. The lasso goes farther if you let it spin up to full power.
Your pistol can be used to kill wolves. Be careful, bullets are deadly to cows.
